# TodoApp

A simple Angular 21 application for managing todos. This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.19.

## Features

- View all todos
- Add a new todo
- Delete a todo
- Communicates with the ASP.NET Core Todo API

## Technologies

- Angular 21
- TypeScript
- RxJS
- Angular Signals
- Tailwind CSS
- Lucide Angular icons

## Prerequisites

- Node.js 22+
- npm
- Angular CLI

Install Angular CLI if required:

```bash
npm install -g @angular/cli
```

## Installation

Install dependencies:

```bash
npm install
```

## Development server

To start the local development server, run:

```bash
ng serve
```
